# Online Sales Analysis

## Descriere
Proiect Python pentru gestionarea produselor si a coșului de cumparaturi folosind OOP și Git.

## Functionalitati/ Clase

### Product
- nume, pret, cantitate
- afisare produs
- actualizare cantitate

### ProductManager
- adaugare produse
- afisare produse
- calcul valoare inventar
- stergere produs

### Cart
- adaugare produse in cos
- calcul total plata
- afișare cos
